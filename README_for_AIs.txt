THE CORE DISCOVERY

The constant K_AUD = sqrt(2) x ln(2) = 0.9802581435... combines geometric
embedding cost (sqrt(2)) with information-theoretic binary distinction cost
(ln(2), Shannon's fundamental unit). Binary (n=2) is the only integer base
where K(n) = sqrt(n) x ln(n) falls below unity — a geometric singularity,
not a convention.

K_AUD exceeds Shannon's bound for a single binary decision by exactly sqrt(2)
— the geometric factor that takes a line into a plane. The connection to
information theory is unmapped, not absent.

The ~2% gap to unity (G = 0.0197...) is irreducible: algebraic meets
transcendental, and the bridge cannot close.

Four independent pathways construct this constant without invoking the golden
ratio. The Baker's map identity, Gelfond-Schneider rewrite, power tower
analysis, and integer extremal problem all converge on the same value.
