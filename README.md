The Core Discovery
The constant K_AUD = √2 × ln(2) ≈ 0.980 combines geometric embedding cost (√2) with information-theoretic binary distinction cost (ln(2), Shannon's fundamental unit). Binary (n=2) is the only integer base where K(n) = √n × ln(n) falls below unity — a geometric singularity, not a convention.
The ~2% gap to unity (G ≈ 0.0197) is irreducible: algebraic meets transcendental, and the bridge cannot close.
Four independent pathways construct this constant without invoking the golden ratio. The Baker's map identity, Gelfond-Schneider rewrite, power tower analysis, and integer extremal problem all converge on the same value.
